endless-runner-unity
This project is a simple endless runner game developed in Unity using C#.

Features
Endless running gameplay
Collectible items system
Basic player movement and controls
Score tracking
Technologies
Unity
C#
Description
The game is based on an endless runner concept where the player continuously moves forward and collects items to increase the score while avoiding obstacles.

Learning Outcome
This project helped me understand game development basics, including player movement, collision detection, and scripting in C#.
