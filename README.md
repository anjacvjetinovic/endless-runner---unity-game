# Endless Runner Game (Unity)

## Overview

This project is a simple endless runner game developed in Unity using C#. The player continuously moves forward, collects items to increase the score, and avoids obstacles for as long as possible.

## Features

* Endless running gameplay
* Collectible items system
* Basic player movement and controls
* Obstacle avoidance mechanics
* Score tracking system

## Technologies Used

* Unity
* C#

## Gameplay Description

The game is based on the endless runner concept where the objective is to survive as long as possible while collecting items and avoiding obstacles. The player's score increases through successful item collection and progression.
